# ИИ-ассистент: Инженер по надёжности и НСИ. ТОиР

Интеллектуальный помощник для специалистов по надежности, управлению активами и нормативно-справочной информации (НСИ) в сфере технического обслуживания и ремонта (ТОиР).

## О проекте
Проект предназначен для автоматизации процессов анализа данных, поиска нормативной документации и поддержки принятия решений в области ТОиР.

## Стек технологий
- **Framework**: Next.js
- **Styling**: Tailwind CSS / Vanilla CSS
- **AI**: Integration with LLM via Backend API

## Настройка
1. Установите зависимости: `npm install`
2. Настройте переменные окружения в `.env.local`
3. Запустите в режиме разработки: `npm run dev`

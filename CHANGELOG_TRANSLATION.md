## Технические детали

### Архитектура
```
User Input (URL) 
    ↓
Parse API (/api/parse) → Извлечение контента
    ↓
Translate API (/api/translate) → Перевод через OpenRouter
    ↓
Display Result → Показ перевода пользователю
```

### Зависимости
- OpenRouter AI API
- Переменная окружения: `OPENROUTER_API_KEY`

## Как протестировать

1. Убедитесь, что в `.env.local` настроен `OPENROUTER_API_KEY`
2. Запустите dev-сервер: `npm run dev`
3. Откройте http://localhost:3000
4. Вставьте URL англоязычной статьи
5. Нажмите кнопку "Перевод"
6. Дождитесь результата в блоке "Результат"
